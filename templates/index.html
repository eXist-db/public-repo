Public Application Repository

This server hosts a public repository of application packages which can be installed into an eXist-db database instance. You can install any package by:

  • Downloading the package and uploading it to an eXist-db database via the Dashboard's Package Manager app.
  • Install the package directly from the Dashboard's Package Manager app. (By default the Package Manager is configured to access the eXist-db public application repository feed, at https://exist-db.org/exist/apps/public-repo/public/apps.xml. To check or edit the repository URL your Package Manager is using, go to your Dashboard, click on the gear icon beneath the eXist-db icon, and you will see a field, "Public Repository URL.")

To keep up with updates to the apps you have installed, visit the Dashboard's Package Manager app. Also, subscribe to the atom feed for all package updates for this repository.
